# Physix Lab

It is an interactive physics learning platform designed to make physics concepts easier by real time calculators, visual simulations and study resources.

---

## Features
1- Some Physics Calculators
2- Some Physics Simulations
3- Some Physics Resources

---

## Built using
- HTML
- CSS
- JS
- CANVAS
- LENIS
